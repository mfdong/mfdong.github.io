Lead Locator (code, design)

As part of my work in The Washington Post’s election lab, I designed and developed an internal website that uses voter registration data to suggest newsworthy locations to political reporters. I used Node and D3 to make geojson data into locator maps/SVGs. Here are some code snippets:

Then, I used Sketch to combine those newsworthy location suggestions into ‘tip sheet’ web pages for journalists:

tip sheet example

I later used React to automate the creation of these tip sheets. I wrote up my process in the “Tip Sheet Generation” and “User Interface Design” sections of this research paper, to be presented at the 2020 C+J conference.

Election Pages (code, design)

Designed and developed React components to be dynamically rendered with live data, published on The Washington Post’s website.

I first conducted design research on other election pages. Then, I wrote React components using Storybook.js, imported data from S3, and rendered them with Next into mobile AMP and desktop pages. This resulted in a few election pages, as well as a document of research insights presented to the Post’s design team.
